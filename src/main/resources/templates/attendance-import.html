Excel 格式要求

  • 第一行必须是表头。
  • 推荐字段：学生姓名、用户名、课程、打卡时间、签退时间、打卡IP、状态。
  • 用户名必须已存在于 user 表。
  • 课程名称必须已存在于 course 表。
  • 打卡时间不能为空，建议格式：yyyy-MM-dd HH:mm:ss。
  • 签退时间可以为空，也可以填写“未签退”。
